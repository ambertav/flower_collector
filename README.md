# FlowerCollector

A web application that allows the user to collect flowers, maintain a watering log for each flower, and add various types pollinators that can pollinate each flower.

## Technologies Used

- Django
- Python
- PostgreSQL
- Amazon Web Services
- HTML5
- CSS3
- Bulma
- Heroku
